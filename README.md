<h1>AP Physics 1 Quiz</h1>

<p>A simple command-line multiple-choice quiz game built in Python, covering core AP Physics 1 equations</p>

## Description

This project presents a set of physics questions in random order, tracks the player score, and offers the option to play again.

## Features

- Multiple-choice questions stored as a list of dictionaries
- Randomized question order each playthrough
- Score tracking
- Replay option after each quiz

## How to Run

1. Download Physics Quiz_Capstone.ipynb from the repository
2. Open the notebook in [Google Colab](https://colab.research.google.com/).
3. Make sure all the cells run (or do Shift + Enter)
4. When prompted, follow the on-screen instructions and enter the letter (A, B, C, or D) corresponding to your answer in the input box that appears below the cell.

## Example

```
Which equation gives average velocity?
A: v = dx/dt
B: v = at
C: v = dx/t^2
D: v = 2dx/t
Choose the correct option: A
Correct!
```
